# Claude Skills for Dotra Medicine Tracker

This directory contains AI assistant skills for the Dotra Medicine Tracker project.

## Installed Skills

### Expo Skills (from github.com/expo/skills)
- **building-native-ui** - Native UI component development
- **expo-api-routes** - API routes setup and patterns
- **expo-cicd-workflows** - CI/CD for Expo apps
- **expo-deployment** - Deployment workflows
- **expo-dev-client** - Development client setup
- **expo-tailwind-setup** - Tailwind CSS integration
- **expo-ui-jetpack-compose** - Jetpack Compose UI
- **expo-ui-swift-ui** - SwiftUI integration
- **native-data-fetching** - Data fetching patterns
- **upgrading-expo** - Expo upgrade workflows
- **use-dom** - DOM usage in Expo

### Callstack Skills (from github.com/callstackincubator/agent-skills)
- **react-native-best-practices** - Performance optimization (JS, native, bundling)
- **react-native-brownfield-migration** - Incremental RN/Expo adoption
- **upgrading-react-native** - RN upgrade workflow
- **github** - PR workflows, code review, branching
- **github-actions** - RN simulator/emulator build artifacts

## Usage

When working with Claude Code or other AI assistants on this project,
these skills provide context-aware guidance for:

- React Native performance optimization
- Expo-specific development patterns
- CI/CD workflows
- Code review and best practices

To reference a skill, ask the AI to check the relevant SKILL.md file.
